Roadmap

DartVN is still in early stages of development. The obvious downside is that many of the features required for building the targetted types of experiences are forthcoming. The upside is that you can weigh in on the included features and the order of implementation. Better yet, if you have any programming experience or want to learn, grab your half-full glass, fork the DartVN repository, and start contributing!

So, where are we heading?

Below is a mid-level list of implemented and planned features.

Implemented

  • YAML (superset of JSON) parsing/processing. Data-api for loading script from URL or textarea id.
  • Stage options (scaling, gridsize), layers, positions, and masks
  • Solid color, images, and sprite sheets
  • Position and script entry/exit of image assets
  • Custom shapes with configurable borders, fill, and alpha ([rounded] rectangle, ellipse)
  • Transition effects: fade, fade through color, crossfade, fade across, slide, pan, and scale
  • Text display, format, alignment, wrapping
  • Script progression options: user input, wait a number of seconds, auto-advance
  • Label and Goto functionality for jumping in a script
  • Modifying an already placed object
  • Basic audio playback
  • Script aliases for repeated commands
  • Within-string emphasis/formatting (text tags)

Planned

  • Easy/efficient character dialog, character format defaults
  • Text animation/words per minute
  • Improved audio control - looping, pausing, crossfading
  • Asset loading indicator
  • Intelligent local caching/storage/"installation"
  • User interactive options/decisions
  • Storing/retrieving user responses
  • Setting script conditions/branching based on decisions
  • String interpolation of variables
  • Saving/loading/storage of user data file
  • Auto-tracking of dialog/scenes encountered
  • Built-in dialog log
  • Built-in menu interface for game options
  • Configurable POST url for cloud-sync
  • Basic interactive objects: hover, touch, click, drag
  • Context-switching API for puzzles/advanced user interactions
